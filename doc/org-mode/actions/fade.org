* Fade Action - #fade {}

  Call JQuery's [[http://docs.jquery.com/Effects/fadeOut][fadeOut(speed)]] on the target element.


** Usage

#+BEGIN_SRC erlang
   wf:wire(myDiv, #fade { speed=500 })
#+END_SRC

** Attributes

   + speed - (/integer/) :: Speed of the effect, in milliseconds.
