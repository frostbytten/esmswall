* Span Element - #span {}

  The span element produces an HTML span.

** Usage

#+BEGIN_SRC erlang
   #span { class=description, text="Some text." }
#+END_SRC

** Attributes

   + text - (/string/) :: The text to display.

   + html\_encode - (/boolean/) :: Set to true to safely html-encode the displayed text.
