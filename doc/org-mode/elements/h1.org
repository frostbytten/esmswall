* H1 Element - #h1 {}

The H1 element produces an HTML heading 1.

** Usage

#+BEGIN_SRC erlang
   #h1 { text="Headline 1" }
#+END_SRC

** Attributes

   + text - (/string/) :: The text of the heading.
